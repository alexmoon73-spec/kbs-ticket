🔥 Firebase 설정 방법
1
console.firebase.google.com → "프로젝트 만들기" → 이름 입력 → 생성
2
좌측 메뉴 빌드 → Realtime Database → "데이터베이스 만들기" → 위치: asia-southeast1 → 테스트 모드 선택
3
좌측 상단 ⚙️ 프로젝트 설정 → 내 앱 → 웹 앱 </> 추가 → 닉네임 입력
4
표시되는 firebaseConfig 값 복사 → 이 HTML 파일 상단 FIREBASE_CONFIG 부분에 붙여넣기 후 저장
5
수정된 HTML 파일을 Netlify에 다시 업로드 → 완료 🎉
